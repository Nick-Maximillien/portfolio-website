🔍 Overview

This decentralized voting application (DApp) was originally built for agricultural cooperative societies, empowering members to vote on critical decisions securely and transparently...

💡 Problem

Traditional cooperative societies often lack transparent, secure voting mechanisms...

  • Ensures only authorized members vote.
  • Enables vote delegation if a member is unavailable.
  • Provides real-time vote results in a visual dashboard.
  • Operates with complete transparency and immutability.

✅ Solution

  • Smart Contract on Ethereum (Sepolia)...
  • Next.js frontend...
  • Real-time charts via Recharts...
  • Wallet Integration...
  • Role Control and Delegation...
  • Live Blockchain Syncing...
