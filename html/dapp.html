🔍 Overview

This decentralized voting application (DApp) was originally built for agricultural cooperative societies, empowering members to vote on critical decisions securely and transparently. Designed for adaptability, this project can easily serve NGOs, schools, unions, SACCOs, HOAs, or any democratic organization—making it a powerful and multidisciplinary blockchain solution.

Built with a modern frontend (Next.js) and a robust smart contract backend (Solidity on Ethereum's Sepolia testnet), the system enforces fair governance, eliminates vote manipulation, and offers an interactive data visualization layer to display results in real-time.

💡 Problem

Traditional cooperative societies often lack transparent, secure voting mechanisms. Physical voting is inefficient, error-prone, and sometimes manipulated. Existing digital solutions are centralized, raising trust and integrity concerns.

This cooperative wanted a blockchain-based alternative that:

  • Ensures only authorized members vote.
  • Enables vote delegation if a member is unavailable.
  • Provides real-time vote results in a visual dashboard.
  • Operates with complete transparency and immutability.

✅ Solution

We created a secure Voting DApp featuring:

  • Smart Contract on Ethereum (Sepolia) to manage all voting logic and proposal handling.
  • Next.js frontend to ensure a responsive and accessible voting experience.
  • Real-time charts (via Recharts) to visualize the vote count.
  • Wallet Integration using Metamask and Ethers.js to authenticate voters.
  • Role Control where only the chairperson can authorize members to vote.
  • Vote Delegation so users can transfer their voting rights.
  • Live Blockchain Syncing to fetch proposals and votes as they happen.

🔐 Key Features

  • Wallet Authentication via Metamask.
  • Role-Based Voting Rights: Only the chairperson can assign voting rights.
  • Proposal Voting: Each member can vote for their preferred option.
  • Delegation Logic: Users can delegate their vote to another member.
  • Live Vote Tracking: Users view results as a bar graph updated in real-time.
  • Error Handling & User Feedback: Integrated toast notifications guide users through each action.
