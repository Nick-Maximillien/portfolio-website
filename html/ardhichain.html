Overview

ArdhiChain is a decentralized land registry system deployed on the Ethereum blockchain. It empowers authorized admins to mint land title NFTs containing verified metadata such as location, title ID, and parcel size. These NFTs can be securely transferred via smart contracts, enabling transparent, tamper-proof land management.

Tech Stack

  • Smart Contract: Solidity using OpenZeppelin (ERC721)
  • Frontend: Next.js & Ethers.js
  • Wallet Integration: MetaMask (via ethers BrowserProvider)
  • Deployment: Infura + Sepolia Testnet
  • Storage: IPFS for token metadata
