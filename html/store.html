Smart Agrovet Webstore

Empowering modern farmers through e-commerce and agritech integration

🌐 Live Site 📦 GitHub Repo

📌 Project Overview

Smart Agrovet is a full-featured online shop for farming inputs and implements, built to streamline agricultural product access for small to large-scale farmers. It’s also designed to integrate with an AI-powered agritech platform featuring a YOLOv8-trained crop disease detector and GPT-4 insights engine — making it not just a shop, but a gateway to intelligent farming.

🎯 Goals

  • Create a modern, mobile-friendly agricultural e-commerce store
  • Enable farmers to browse, search, and buy tools, seeds, fertilizers, and equipment
  • Serve as a frontend gateway for future integration with AI-powered crop diagnosis and insights tools

🧰 Tech Stack

  • Frontend: Next.js, TypeScript, Tailwind CSS
  • Backend: Django with REST API
  • Image Storage: Cloudinary
  • CMS: Django Admin
  • Deployment: Netlify (frontend), Railway (backend)

👨‍💻 My Role

I worked as the sole fullstack developer — designing, coding, and deploying the entire system end-to-end, including:

  • Planning database schema and Django models
  • Designing and building the responsive Next.js frontend
  • Setting up Cloudinary for optimized image delivery
  • Building the admin CMS for product management
  • Deploying and connecting the frontend and backend via RESTful APIs

🚧 Challenges

One of the key challenges was optimizing image delivery for product pages while keeping load times low for rural users with limited connectivity. I resolved this by integrating Cloudinary with dynamic image scaling. Another challenge was structuring a backend that could eventually scale into a more advanced agritech platform.

📈 Outcome & Lessons Learned

The Smart Agrovet Webstore is now a solid foundation for a growing agritech ecosystem. I learned a lot about REST API integration, performance optimization, and designing systems with future AI scalability in mind. This project also improved my ability to juggle both frontend and backend development independently.
